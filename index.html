The Ancient Greek and Latin Dependency Treebank (AGLDT) is the earliest treebank for Ancient Greek and Latin. The project started at Tufts University and is currently under development and maintenance at Leipzig University-Tufts University. Data and full documentation are made freely available on GitHub under a CC BY-SA 3.0 Licence. The present webpage is for presentational purposes only. More information about the creation of the data is contained in the subfolders of the GitHub repository. The current release is v. 2.0.

The Ancient Greek Dependency Treebank

The AGDT 2 has been created as a refinement of the AGDT 1. In a new extended annotation scheme, it defines the morphological and syntactic annotations more stringently, and adds a semantic layer built on the categories identified in H. W. Smyth's Grammar. Aesop and Pseudo-Apollodorus are fully compliant with the new annotation scheme and have been annotated for semantics. The texts available in the current release (v. 2.0) are the following:
